# r3f-boilerplate-vite

react-three-fiber, vite, glsl shader support, react-spring, and react-three-a11y

### Installation

```
git clone https://github.com/apttap/r3f-boilerplate-vite.git
cd r3f-boilerplate-vite
npm install
```

### Running

```
npm run dev
```

### Building

```
npm run build
```
